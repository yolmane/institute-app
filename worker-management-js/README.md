# Система управления персоналом АНО "Институт инноваций права"

## Описание
Приложение для управления данными работников организации, реализованное на чистом JavaScript с использованием ООП принципов.

## Функциональность
1. Добавление новых работников с валидацией данных
2. Поиск работников по стажу
3. Сортировка работников
4. Статистика по персоналу
5. Сохранение данных в LocalStorage
6. Управление данными через интуитивный интерфейс
